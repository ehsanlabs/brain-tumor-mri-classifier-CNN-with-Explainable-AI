# 🧠 Brain Tumor MRI Classifier

A deep learning project that classifies brain tumor MRI scans into 4 categories with **93% accuracy** using Convolutional Neural Networks (CNN).

## 🎯 Tumor Classes
| Class | Description |
|-------|-------------|
| 🔴 Glioma | Most common primary brain tumor |
| 🟡 Meningioma | Arises from brain membranes |
| 🟣 Pituitary | Forms in pituitary gland |
| 🟢 No Tumor | Normal MRI scan |

## 🛠️ Tech Stack
- Python, TensorFlow, Keras
- OpenCV, NumPy, Pandas
- Streamlit (Web App)
- CNN Architecture

## 📊 Model Performance
- **Accuracy: 93%**
- 4-class classification
- Trained on MRI brain scan dataset
