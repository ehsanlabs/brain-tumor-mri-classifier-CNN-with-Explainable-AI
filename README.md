# 🧠 Brain Tumor MRI Classifier

A deep learning project that classifies brain tumor MRI scans into 4 categories 
with 93% accuracy using Convolutional Neural Networks (CNN).

## 🎯 Classes
- Glioma
- Meningioma  
- Pituitary Tumor
- No Tumor

## 🛠️ Tech Stack
- Python, TensorFlow, Keras
- OpenCV, NumPy
- Streamlit (Web App)

## 🚀 Run the App
pip install -r requirements.txt
streamlit run app.py

## 📊 Results
- Accuracy: 93%
- Dataset: MRI brain scan images
